

1.- Para que valide arreglos

campo: z.array(z.string()) <--se especifica el tipo de dato que se espera dentro del arreglo

ejemplo de un arreglo recibido:
{
    "tags": ["juan", "maria", "pedro"]
}



2.- Para validar arreglos de objetos
campo: z.array(z.object({campo: z.string()})) <--se especifica el tipo de dato que se espera dentro del arreglo

ejemplo de un arreglo recibido:
{
    "tags": [
        { "persona": "juan" },
        { "persona": "maria" },
        { "persona": "pedro" }
    ]
}


3.- el .refine

 se usa para validaciones personalizadas que involucran múltiples campos del objeto

 3.1 ejemplos: "Validar que tenga email O usuario"

 .refine(data => data.email || data.usuario, {
    message: 'Debe proporcionar email o usuario',
    path: ['email']
})

 3.2 ejemplos: "Validar que la nueva contraseña sea diferente a la actual"

 .refine(data => data.password !== data.newPassword, {
    message: 'La nueva contraseña debe ser diferente a la actual',
    path: ['newPassword']
})
